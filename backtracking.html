Sudoku

Joacă Reguli Backtracking

În 2014, în Admiterea de Informatică de la Universitatea "Alexandru Ioan Cuza" din Iași s-a dat următoarea problemă:

Pbinfo Subiect

O grilă SUDOKU este o matrice 9 x 9 care respectă următoarele proprietăți:

  1. fiecare element al matricii este un număr natural între 1 la 9;
  2. fiecare linie conține toate numerele naturale de la 1 la 9;
  3. fiecare coloană conține toate numerele naturale de la 1 la 9;
  4. fiecare dintre cele 9 submatrici de dimensiune 3 x 3 conține toate numerele de la 1 la 9;

Un puzzle SUDOKU este o matrice 9 x 9 completată parțial cu numere naturale de la 1 la 9. O soluție a unui astfel de puzzle este o grilă SUDOKU care coincide cu puzzle-ul pe pozițiile precompletate.

Cerința

Scrieți un program care, pentru o matrice 9 x 9 dată, reprezentând un puzzle SUDOKU, determină o soluție a unui astfel de puzzle, după regulile de mai sus.

Date de intrare

Fișierul de intrare sudoku.in conține o matrice 9 x 9 (câte 9 elemente pe câte o linie a fișierului, fiecare element fiind separat printr-un spațiu), reprezentând un puzzle SUDOKU , în care pozițiile necompletate au valoarea 0.

Date de ieșire

Fișierul de ieșire sudoku.out va conține o matrice 9 x 9 (câte 9 elemente pe câte o linie a fișierului, fiecare element fiind separat printr-un spațiu), reprezentând soluția puzzle-ului din fișierul de intrare.

Restricții și precizări

  • puzzle-ul din fișierul de intrare are soluție unică, pentru toate testele

Explicație

Pentru a rezolva această problemă, trebuie să folosim un algoritm de backtracking, care va încerca să completeze celulele goale cu o cifră de la 1 la 9 care este validă. O cifră poate fi pusă într-o celulă dacă nu mai apare încă o dată pe aceeași linie, coloană sau submatrice 3x3. Dacă la un moment dat nu există nicio cifră validă, algoritmul revine la pasul anterior și încearcă altă valoare, repetând procesul de backtracking. Când toate celulele sunt completate corect, am găsit soluția și oprim algoritmul.

Cod

#include <fstream>
#include <iostream>
using namespace std;
ifstream fin("sudoku.in");
ofstream fout("sudoku.out");

int a[10][10], ok = 0;

int valid(int k, int row, int col) {
  for (int i = 1; i <= 9; i++)
    if (a[row][i] == k)
      return 0;
  for (int i = 1; i <= 9; i++)
    if (a[i][col] == k)
      return 0;

  int startRow = (row - 1) / 3 * 3 + 1;
  int startCol = (col - 1) / 3 * 3 + 1;
  for (int i = startRow; i <= startRow + 2; i++)
    for (int j = startCol; j <= startCol + 2; j++)
      if (a[i][j] == k)
        return 0;
  return 1;
}

void afis() {
  for (int i = 1; i <= 9; i++) {
    for (int j = 1; j <= 9; j++)
      fout << a[i][j] << ' ';
    fout << endl;
  }
}

void bkt(int row, int col) {
  if (ok)
    return;
  if (row == 10) {
    afis();
    ok = 1;
    return;
  } else if (col == 10) {
    bkt(row + 1, 1);
  } else if (a[row][col] != 0) {
    bkt(row, col + 1);
  } else {
    for (int k = 1; k <= 9; k++)
      if (valid(k, row, col)) {
        a[row][col] = k;
        bkt(row, col + 1);
        a[row][col] = 0;
      }
  }
}

int main() {
  for (int i = 1; i <= 9; i++)
    for (int j = 1; j <= 9; j++)
      fin >> a[i][j];
  bkt(1, 1);
  return 0;
}
